# Enterprise AI Application Portal

企业级 AI 应用门户 / 智能工作入口项目。

当前项目已经从“传统门户”方向扩展为“对话式统一入口 + 内部智能体执行 + 企业流程协同”的产品形态，适用于企业内部会议、采购、制度问答、周报、差旅、权限开通等场景。

## 项目结构

- `DEMO/demo-a-c/`：当前可直接演示的高保真静态 demo
- `PROJECT_DOCS/`：主线项目文档
- `PROJECT_DOCS/archive/`：归档材料
- `GPT_doc/`：持续进展记录
- `企业AI门户.png`：原始门户原型图
